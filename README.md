# 🌿 Lestari Care Management App

Aplikasi manajemen klinik modern yang dibangun untuk membantu pengelolaan data medis, keuangan, karyawan, hingga produk dan obat.  
Didesain dengan fokus pada **efisiensi, kemudahan penggunaan, dan keamanan data**.

---

## 🚀 Tech Stack
- **[Next.js](https://nextjs.org/)** – React framework untuk frontend & backend
- **[Chakra UI](https://chakra-ui.com/)** – Komponen UI modern & responsif
- **[Prisma](https://www.prisma.io/)** – ORM untuk akses database yang aman & cepat
- **[Supabase](https://supabase.com/)** – Backend as a Service (auth, database, storage)

---

## ✨ Features
- 🩺 **Management Rekam Medis Pasien**  
  Simpan & kelola data rekam medis pasien secara aman.
- 👥 **Management User/Admin**  
  Sistem autentikasi & otorisasi berbasis role.
- 💰 **Pencatatan Keuangan**  
  Catat transaksi pemasukan & pengeluaran secara detail.
- 👨‍⚕️ **Management Karyawan**  
  Kelola data staf dan dokter dengan mudah.
- 📦 **Management Produk**  
  Monitoring inventaris produk.
- 💊 **Manajemen Obat**  
  Catat & atur stok obat dengan efisien.
- 📊 **Rekapitulasi Pemasukan/Pengeluaran**  
  Laporan keuangan otomatis & mudah dipahami.
